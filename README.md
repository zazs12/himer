
# 家用物品管家（增强版）

## ✨ 功能亮点
- **完整 CRUD**：增、删、改、查物品
- **多级位置**：房间 → 容器 → 层/格 → 具体描述
- **快速输入**：常用房间/容器自动记忆
- **统计分析**：
  - 按房间柱状图
  - 按分类饼图
  - 容器 Top 5 分布
- **PWA 支持**：可安装、离线使用
- **iCloud 同步**：导出/导入 JSON

## 🚀 部署
1. 创建 GitHub 仓库（如 `home-items-manager`）
2. 上传以上 4 个文件
3. 启用 GitHub Pages（Settings → Pages）
4. 在 iPhone Safari 中打开 → 分享 → “添加到主屏幕”

> 数据仅存于本机，隐私安全。定期导出 JSON 备份！

# 家用物品管家 (PWA)

✅ 搜索 + 分类  
✅ 拍照/相册上传  
✅ 条码扫描（需 HTTPS）  
✅ iCloud Drive 手动同步（JSON 导入/导出）  
✅ iPhone 完美适配

## 如何使用

1. 访问：`https://<your-github-username>.github.io/home-items-manager-pwa/`
2. 在 Safari 中点击「分享」→「添加到主屏幕」
3. 导出数据 → 保存到 iCloud Drive → 其他设备导入

## 条码扫描

仅在 GitHub Pages（HTTPS）下可用，本地 file:// 无法使用摄像头。
